Generate a rating report card based on different criteria.

== Description ==

Rating Report allows you to generate a rating report card based on different criteria. Use it to review books, movies, or products.

== Installation ==

1. Upload `rating-report` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Adjust the settings in Settings -> Rating Report
